# KidGuard Chrome Extension

## Installation Instructions

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select this `chrome-extension` folder
5. The extension icon should appear in your toolbar

## Setup

1. Click the KidGuard extension icon
2. Enter your Parent ID (from the parent dashboard)
3. Enter your child's name
4. Click "Save Settings"

## Usage

The extension automatically monitors Roblox chat when your child visits roblox.com. It will:
- Detect grooming risk patterns in real-time
- Alert you via SMS for high-risk incidents
- Download evidence files to your Downloads folder
- Log all activity for review in the parent dashboard
